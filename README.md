# 🤖 Lucía - Asistente Virtual en Python

## 📌 Descripción
Lucía es un asistente virtual desarrollado en Python como mi primer proyecto de programación. Su objetivo fue aplicar conceptos fundamentales del lenguaje mediante un programa interactivo capaz de almacenar información del usuario y ofrecer distintas funcionalidades desde una interfaz de consola.

El proyecto implementa un sistema de registro e inicio de sesión utilizando archivos JSON para guardar usuarios y preferencias, permitiendo que la información permanezca disponible entre distintas ejecuciones del programa. Una vez iniciada la sesión, el asistente recuerda datos personales del usuario, como su nombre, color favorito y canción favorita, creando una experiencia más personalizada.

Además de administrar usuarios, el asistente incorpora diversas herramientas y minijuegos, como consultar la fecha y la hora actuales, jugar Piedra, Papel o Tijera, Cara o Cruz y Adivina el Número, así como generar tablas de multiplicar.

Este proyecto representa mi primer acercamiento a la programación estructurada, el manejo de archivos, la modularización mediante funciones y la persistencia de datos en Python.

## 🔧 Tecnologías
- Python 3
- JSON
- Módulo datetime
- Módulo random

## 📝 Funciones 
1. **👤 Registro e inicio de sesión de usuarios.**
2. **💾 Almacenamiento de información mediante archivos JSON.**
3. **🧠 Recordatorio de datos personales del usuario.**
4. **🕒 Consulta de fecha y hora del sistema.**
5. **🎮 Minijuegos interactivos:**
- Piedra, Papel o Tijera.
- Cara o Cruz.
- Adivina el Número.
6. **➗ Generación de tablas de multiplicar.**
7. **🖥️ Interfaz completamente basada en consola.**
